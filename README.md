# This is a Tile Adventure Game made in Haskell
The icons in this game are:
@ - The player
^, V, >, and < - Enemies
S - Score tile
B - Bomb tile
O - Teleporter tile
E - End Goal

You can choosed to move around as long as it is a free space.
You cannot move into an enemy or a wall.

You can also choose to view and manage your scores in the score submenu.

Try to get the highest score possible!
